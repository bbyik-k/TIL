# [Core JS] 1-1. 데이터 타입의 종류와 분류 기준

## 요약

자바스크립트의 데이터 타입 구분은 값이 아니라 **저장 방식과 동작 방식**의 차이다.

## 핵심 내용

- 데이터 타입은 기본형과 참조형으로 나뉜다.
- 기본형은 값 자체가 저장되고, 참조형은 값의 위치가 저장된다.
- 이 차이로 인해 불변성과 가변성처럼 보이는 현상이 발생한다.
- 복제 시
  - 기본형(불변성): 값이 담긴 주솟값을 바로 복제
  - 참조형: 값이 담긴 주솟값들로 이루어진 묶음을 가리키는 주솟값을 복제

## Check Point

- 불변성(immutability)이란?
  - 메모리 지식 배경
  - 식별자와 변수 개념 구분

## 언제 문제가 되는가 / 언제 쓰는가

- React state 업데이트 시 객체를 직접 수정하면 문제가 생기는 이유를 설명해 준다.

## 정리

데이터 타입의 차이는 문법이 아니라 메모리 동작 방식의 차이다.
